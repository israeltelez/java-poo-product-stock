# Java - Introdução à Programação Orientada a Objetos (POO)

Este projeto foi desenvolvido como exercício prático para aprender os conceitos iniciais de Programação Orientada a Objetos em Java.

##  Conceitos aplicados

Classe e objeto
Atributos
Métodos
Encapsulamento (básico)
Método `toString()`

## Funcionalidades

O sistema permite:

 Cadastrar um produto
 Adicionar itens ao estoque
 Remover itens do estoque
 Exibir os dados atualizados do produto

##  Tecnologias utilizadas

 Java
 Eclipse IDE
